This is a complete STM32CubeIDE project that uses the Software library created as part of a research program for low-level rover motion control (conversion and execution of high-level navigation library commands into rover actuator control) and feedback from rover sensors to the navigation and computer vision system, in order to optimize the route during rover movement. Specifically, the STM32 C module motion_lib.h is used. (Core/Inc/motion_lib.h, Core/Src/motion_lib.c)
